# Projet_ACOL

Bonjour à tous,

Bienvenue sur le projet ACOL. 
Ce projet a pour but de créer un petit jeu vidéo afin de s'exercer au principe de conception d'objet et de cahier des charges. On utilisera Java pour ce projet.

Pour lancer le jeu : 
    - Il suffit d'executer "make play" dans votre terminal de commande. 
    
En espérant que le jeu vous plaise !
